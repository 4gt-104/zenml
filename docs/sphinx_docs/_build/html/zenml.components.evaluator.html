Parameters

  • input_dict – Input dict from input key to a list of Artifacts. These are often outputs of another component in the pipeline and passed to the component by the orchestration system.

  • output_dict – Output dict from output key to a list of Artifacts. These are often consumed by a dependent component.

  • exec_properties – A dict of execution properties. These are inputs to pipeline with primitive types (int, string, float) and fully materialized when a pipeline is constructed. No dependency to other component or later injection from orchestration systems is necessary or possible on these values.
