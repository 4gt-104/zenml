Do(input_dict: Dict[str, List[tfx.types.artifact.Artifact]], output_dict: Dict[str, List[tfx.types.artifact.Artifact]], exec_properties: Dict[str, Any]) → None¶

Runs batch inference on a given model with given input examples. :param input_dict: Input dict from input key to a list of Artifacts.

  • examples: examples for inference.

  • model: exported model.

  • model_blessing: model blessing result, optional.

Parameters

  • output_dict – Output dict from output key to a list of Artifacts. - output: bulk inference results.

  • exec_properties – A dict of execution properties. - model_spec: JSON string of bulk_inferrer_pb2.ModelSpec instance. - data_spec: JSON string of bulk_inferrer_pb2.DataSpec instance.

Returns

None
